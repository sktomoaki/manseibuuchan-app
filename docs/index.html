🔧 音声文字起こしエンジンの選び方

「🎤 音声から作成」タブで2種類のエンジンから選べます。

🤖 Groq Whisper完全無料

  • ✅ 高速（30秒〜2分）
  • ✅ 完全無料
  • ✅ 長時間ファイルも自動チャンク分割で対応
  • ✅ 1人の録音・講演・インタビューに最適
  • ❌ 話者識別なし（誰が話したかわからない）

🎯 AssemblyAI会議録音おすすめ

  • ✅ 話者識別あり（誰が話したか自動判定）
  • ✅ 100時間まで無料
  • ✅ 複数人の会議録音に最適
  • ⏱ やや時間がかかる（1〜3分）
場面おすすめエンジン
👤 1人の録音・講演・インタビュー🤖 Groq Whisper（速くて無料）
👥 複数人の会議（誰が話したか残したい）🎯 AssemblyAI（話者識別あり）
⏱ 2〜4時間の長時間録音🤖 Groq Whisper（自動チャンク分割で対応）
✨ 長時間音声も安心： 30分を超える音声は自動的に10分チャンクに分割して処理されます。進捗バーで何チャンク目かリアルタイム表示。

🎤 音声ファイルから議事録を作る手順

「万世ブーちゃんを開く」→「🎤 音声から作成」タブ（最初から開いています）
文字起こしエンジンを選ぶ
  • 会議録音・複数人 → 🎯 AssemblyAI（話者識別あり）
  • 1人の録音・速記・長時間 → 🤖 Groq Whisper
音声ファイルをアップロード

ファイルサイズ・音声の長さ・予想処理時間が自動表示されます。

会議情報を入力

会議日時・タイトル・参加者などを入力します（空欄でも生成できます）。

「▶ 文字起こし＋議事録を作成」ボタンをクリック

自動で文字起こし→議事録生成まで一括処理されます。長時間ファイルはチャンクごとに進捗表示。

議事録・文字起こしデータをダウンロード

HTML形式でダウンロードできます。印刷にも対応しています。

📄 テキストから議事録を作る手順

「万世ブーちゃんを開く」→「📝 テキストから作成」タブをクリック
テキスト欄に文字起こしを貼り付ける

会議の文字起こしテキストを貼り付けます。

参加者・メモを入力して議事録生成
  • 参加者の名前と所属を入力
  • 出力スタイルは「議事録＋文字起こしデータ」がおすすめ
  • 「🐷 議事録を作る」ボタンをクリック

🔧 よくあるトラブル

❌ アプリが「This app has gone to sleep」と表示される

「Wake up this app」ボタンをクリックしてください（1〜2分で復帰します）。

❌ パスワードが違うと言われる

月が変わるとパスワードが変わります。Googleアカウントでログインするか、管理者（田原）にご連絡ください。

❌ 音声ファイルのアップロードに失敗する

対応形式：m4a / mp3 / wav / mp4 / ogg / flac。2〜4時間の長時間ファイルも自動チャンク分割で対応しています。

❌ Groq でエラー「Request Entity Too Large」が出る

⚠️ v2.9以降は25MBを超えるファイルを自動的に5分チャンクに分割して送信します。このエラーは通常発生しなくなりました。もし出る場合はページを更新して再試行してください。

❌ AssemblyAI でエラーが出る

⚠️ AssemblyAI の無料枠（100時間）を超えた可能性があります。Groq Whisper に切り替えるか、システム管理者（田原）にご連絡ください。
